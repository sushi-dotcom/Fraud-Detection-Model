# 📌 Fraud Detection Model

A machine learning-based fraud detection system designed to identify fraudulent transactions using data preprocessing, feature engineering, and classification models.

---

## 🚀 Project Overview
Fraud detection is a critical problem in financial systems where fraudulent transactions are rare but highly impactful. This project builds a robust pipeline to detect fraud by handling class imbalance and applying machine learning algorithms.

---

## 🧠 Key Features
- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing  
- ⚖️ Handling Imbalanced Dataset  
- 🔍 Feature Engineering  
- 🤖 Model Training (Logistic Regression, Random Forest, XGBoost)  
- 📈 Model Evaluation (Accuracy, Precision, Recall, F1-score, ROC-AUC)  
- 📉 Confusion Matrix Visualization
